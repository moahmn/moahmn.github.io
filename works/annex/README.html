annexShow使用说明

head标签引入CSS

<link rel="stylesheet" type="text/css" href="styles/slick.css"/>
<link rel="stylesheet" type="text/css" href="styles/slick-theme.css"/>
<link rel="stylesheet" type="text/css" href="styles/annex.css"/>

在body结束标签前引入JS

<script type="text/javascript" src="script/jquery-1.12.2.min.js"></script>
<script type="text/javascript" src="script/slick.js"></script>
<script type="text/javascript" src="script/mark.js"></script>//可选，在需要显示HTML关键字高亮时引入
<script type="text/javascript" src="script/annexShow.js"></script>

html结构

<div id='annexShowWrapper'>
 <ul>
    <li data-txt="图片" data-url="resource/1.jpg,resource/2.jpg,resource/4.jpg" data-names="图片名1,图片名2,图片名3" data-desc="描述1,描述2,描述3" class="annex-items annex-pic"><a href="javascript:void(0)">图片</a></li>

 </ul>
</div>

<div class="annex-pop clearfix" id="annexPopUp"></div>
属性 描述
data-txt 弹层显示的四种类型：图片、音频、视频、HTML（重要，必须传递）
data-url 需要显示的附件url地址，除了HTML其余三种附件都可以传递多个附件地址，使用","分隔。data-txt类型为HTML的data-url传参格式为：address="需要显示的附件地址"&keyWord="需要高亮的关键字"，请自行解决跨域问题
data-names 附件的名字，可传递多个(使用","分隔)，和data-url属性是一对一的关系，若没有传递，则使用附件url地址的最后一个路径分隔符至结尾的字符串。
data-desc 对图片的描述，该属性只作用于data-txt="图片"的类型,可传递多个(使用","分隔)，和data-url属性是一对一的关系，若没有传递，则描述显示为空

*如果需要显示pdf附件（借助pdf.js）,参考li最后一个节点,（不需要传递任何data参数）只需要a标签的href设置为pdfjs/web/viewer.html?file=pdf附件地址

调用

<script type="text/javascript" >

	}
</script>

配置项

选项 类型 默认值 描述
showImgName Boolean true 是否显示图片的名称，默认显示
showImgDesc Boolean true 是否显示图片描述，默认显示
clickMaskClose Boolean true 是否在点击遮罩层时触发与关闭按钮相同的动作
highLightHtmlPath String "./" 配置highLightHtmlPath.html这个文件的相对位置，默认与demo.html在同一级目录，如果换了位置，请配置正确的相对位置或者绝对位置（相对位置相对于demo.html）
keyWordParticipleUrl String "" 关键字分词接口，如果提供分词接口，则对关键字分词后再进行高亮，否则直接对关键字进行高亮
participleDataKey String "" 接口传递关键字参数的key
annexSlickLayer Object {'width': 600,'height':'auto'} 图片弹层的样式配置选项，请不要配置固定高度，让高度自适应，可传入任何修改样式的属性
annexAudioLayer Object {'width': 400,'height':'auto'} 音频弹层的样式配置选项，可传入任何修改样式的属性
annexVideoLayer Object {'width': 600,'height':'auto'} 视频弹层的样式配置选项，可传入任何修改样式的属性
annexHtmlLayer Object {'width':'60%','height':'auto'} HTML弹层的样式配置选项，可传入任何修改样式的属性
